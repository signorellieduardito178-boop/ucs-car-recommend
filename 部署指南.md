# UCS展车外借系统 — 详细部署步骤（含命令行）

> 前置条件：已完成 MongoDB Atlas 注册 + 设置允许任意IP访问（0.0.0.0/0）

---

## 第一步：获取 MongoDB 连接字符串

### 1.1 回到 MongoDB Atlas 控制台

1. 打开 https://cloud.mongodb.com
2. 进入你的项目 → 点击左侧「Database」→ 找到你的集群

### 1.2 获取连接字符串

1. 点击集群卡片上的 **「Connect」** 按钮
2. 选择 **「Drivers」**
3. 在下拉菜单中选择 **Node.js**，版本选 **5.5 或更高**
4. 复制下方的连接字符串，格式如下：

```
mongodb+srv://ucs_admin:<db_password>@cluster0.xxxxx.mongodb.net/ucs_car?retryWrites=true&w=majority
```

5. **把 `<db_password>` 替换成你实际设置的密码**

   例如，如果你的密码是 `MyPass123`，最终字符串为：
   ```
   mongodb+srv://ucs_admin:MyPass123@cluster0.xxxxx.mongodb.net/ucs_car?retryWrites=true&w=majority
   ```

6. **把这串连接字符串保存到记事本里，等下要用**

## 第三步：部署到 Vercel

### 3.1 注册/登录 Vercel

1. 打开 https://vercel.com
2. 点击 **「Sign Up」** → 选择 **「Continue with GitHub」**
3. 授权 Vercel 访问你的 GitHub 仓库

### 3.2 导入项目

1. 登录后进入 https://vercel.com/dashboard
2. 点击 **「Add New...」** → **「Project」**
3. 在列表中找到 **「ucs-car-recommend」** 仓库，点击 **「Import」**

### 3.3 配置项目

进入项目配置页面后：

| 配置项 | 值 |
|--------|-----|
| Framework Preset | 保持默认（Next.js） |
| Root Directory | `./`（默认） |
| Build Command | 保持默认 |
| Output Directory | 保持默认 |

**不要点 Deploy，先配置环境变量！**

### 3.4 添加环境变量

1. 找到页面上的 **「Environment Variables」** 区域（展开它）
2. 点击 **「Add」** 添加变量：

   **第一行：**
   - Name: `MONGODB_URI`
   - Value: 把你第一步复制的连接字符串粘贴进来

   **第二行（可选，飞书推送）：**
   - Name: `FEISHU_WEBHOOK_URL`
   - Value: 你的飞书机器人 webhook 地址（如果没有就不填）

3. 确认环境变量添加完毕后，点击 **「Deploy」**

### 3.5 等待部署完成

- 部署过程大约 1-2 分钟
- 完成后页面会显示 **「Congratulations!」**
- 复制分配的域名，例如：
  ```
  https://ucs-car-recommend-xxxxx.vercel.app
  ```

## 常见问题排查

### 问题 1：git push 时提示权限错误

```
fatal: unable to access 'https://github.com/...': The requested URL returned error: 403
```

**解决**：GitHub 已不支持密码登录，需要创建 Personal Access Token。

1. 打开 https://github.com/settings/tokens
2. 点击 **「Generate new token (classic)」**
3. 勾选 **repo** 权限
4. 生成后复制 token，用它代替密码登录

### 问题 2：Vercel 部署失败（Build Error）

**解决**：检查 MongoDB URI 是否已正确配置。环境变量必须配置好才能部署成功。

### 问题 3：销售端提示「搜索失败」

**解决**：
1. 确认 MongoDB Atlas 的 Network Access 已添加 `0.0.0.0/0`
2. 确认 Vercel 的 Environment Variables 中 `MONGODB_URI` 正确
3. 确认已上传展车和门店数据
